Stacked Area Chart

Stacked area graphs are useful for displaying the trend of contribution of values of two or more variables to the total over categories or time.

Chart Options

Dimensions

  • width
  • height

Title properties

  • title
  • titlePosition
  • titleSize

Lifecycle

  • renderCallback - executed immediatly before render is called

Legend properties

  • legend (true|false)
  • legendPosition (right, top, bottom, left)
  • legendAlign (right, left, center, top, middle, bottom)
  • legendSize
  • legendDrawLine (true|false)
  • legendDrawMarker (true|false)
  • showTooltips (true|false)

Data properties

  • timeSeries (true| false)
  • timeSeriesFormat. For example: "%Y-%m-%d"

Stacked Area chart properties

  • orientation - horizontal or vertical. Default: vertical
  • showValues - Show or hide line value. Default: false
  • panelSizeRatio - Ratio of the band occupied by the pane. Default: 0.5 (50%)
  • stacked - Stacked? Default: true
  • lineSizeRatio - In multiple series, percentage of inner band occupied by lines. Default: 0.5 (50%)
  • maxLineSize - Maximum size of a line in pixels. Default: 2000

Extension Points

The following list serves as the basis for Stacked Area chart extension points:
  • base_
  • title_
  • titleLabel_
  • xAxis_
  • yAxis_
  • xAxisRule_
  • yAxisRule_
  • xAxisRuleGrid_
  • yAxisRuleGrid_
  • tooltip_
  • chart_
  • area_
  • dot_
  • lineDot_
  • lineLabel_
Sample extension points:
  • base_fillStyle
  • titleLabel_font
  • area_fillStyle
